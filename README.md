## Project Overview

The Student Reference Portal is a database-driven web application developed for the IT6037 Project.

The system provides educational reference materials for students studying:

- Art
- Mathematics
- Technology

The application allows different users to perform different tasks based on their role.

---

## User Roles

### Student
- Browse articles by category
- Search articles by keyword

### Tutor
- Add articles
- Modify articles

### Administrator
- Add articles
- Modify articles
- Delete articles

## Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Import the SQL database.
4. Copy the project into the `htdocs` folder.
5. Open:

http://localhost/Student_Reference_Portal/
